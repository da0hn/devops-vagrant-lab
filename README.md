# devops-vagrant-lab

### Instalation

* Virtual Box 
* Vagrant
* vagrant-vbguest plugin
  * `vagrant plugin install vagrant-vbguest --plugin-version 0.21`


### Commands

```sh
vagrant up
```

* To initialize VMs use command above in folders
  * `/ansible-lab/app01`
  * `/ansible-lab/control-node`
  * `/ansible-lab/db01`

```sh
vagrant halt
```

* To shutdown VM use command above
